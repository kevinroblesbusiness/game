# 🚀 Quick Start Guide

New to the project? Start here!

## 📖 For Players

Just want to play the game?

1. **Open the game**: Double-click `starry-valley-ch2.html`
2. **Read the controls**: Use arrow keys to move, T to talk
3. **Start exploring**: Collect gems and talk to NPCs
4. For full guide, see [GAMEPLAY.md](./docs/GAMEPLAY.md)

## 👨‍💻 For Developers

Want to modify or contribute to the game?

### 1️⃣ Read the Code Overview
- Open [CLAUDE.md](./CLAUDE.md)
- Understand the file structure and key functions
- ~10 minutes to read

### 2️⃣ Explore the Code
- Open `starry-valley-ch2.html` in your editor
- Look for comments like `// tile floor`, `// update player`
- Jump to interesting functions

### 3️⃣ Make a Small Change
Try one of these beginner tasks:
- Change a color in the CSS (lines 1-100)
- Add a comment to explain a function
- Modify NPC dialogue (search for `DIALOGS`)

### 4️⃣ Test Your Change
1. Save the file
2. Refresh your browser (F5)
3. See the change in action

### 5️⃣ Commit Your Work
```bash
git add starry-valley-ch2.html
git commit -m "Description of your change"
git push origin feature/your-name
```

## ✅ Sanity Checks

Before committing changes:

```
Does your code:
✓ Work in the browser?
✓ Have no console errors? (F12 → Console tab)
✓ Follow the code style?
✓ Have clear commit message?
```

If yes to all, you're ready to push!
