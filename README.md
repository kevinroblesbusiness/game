# Starry Valley - Chapter 2

A pixel-art adventure game where you play as Angie exploring Starry Valley to solve mysteries and collect clues.

## 🎮 How to Play

1. Open `starry-valley-ch2.html` in your web browser
2. Use **Arrow Keys** to move around the map
3. Press **T** or **Space** to talk to NPCs and interact with objects
4. Collect gems and uncover clues to progress through the level
5. Use number keys **1-4** to select different tools:
   - **1**: Hand (collect gems, interact)
   - **2**: Shield (block attacks)
   - **3**: Crystal Staff (attack boss)
   - **4**: Journal (view clues and story)
